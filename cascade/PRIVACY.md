# Privacy Policy — Cascade: Tab & Bookmark Organizer

**Last updated: March 2026**

## Overview

Cascade is a Chrome extension that displays your open tabs and bookmarks in a side panel, organized by domain in a clean tree view.

## Data Collection

**Cascade does not collect, transmit, or share any personal data.**

This extension processes tab and bookmark information entirely within your browser. No data is stored to disk or sent externally.

## What the Extension Accesses

| Access | Purpose |
|---|---|
| Open tabs (titles and URLs) | To build the domain-grouped tree view |
| Bookmarks | To display your bookmarks in the tree view |
| Favicons | To show website icons next to each item |
| Recently closed sessions | To enable the "Undo" feature for restoring closed tabs |
| Side panel | To display the tree view in Chrome's side panel |

## What the Extension Does NOT Do

- Does not send any data to external servers
- Does not track browsing activity
- Does not use analytics or advertising services
- Does not store any data to disk or cloud storage
- Does not use cookies
- Does not require account creation or login

## Data Storage and Deletion

Cascade does not persist any data. The tab and bookmark tree is generated dynamically each time the side panel is opened or updated. Uninstalling the extension removes it completely with no residual data.

## Changes to This Policy

If this policy is updated, the "Last updated" date at the top of this page will be revised. Continued use of the extension after changes constitutes acceptance of the updated policy.
