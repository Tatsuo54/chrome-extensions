# Cascade: Tab & Bookmark Organizer

開いているタブとブックマークをサイドパネルにドメインごとのツリー表示で整理するChrome拡張機能です。

## 主な機能

### 🌳 ドメイン別ツリー表示
開いているタブをドメインごとに自動グループ化し、階層的なツリーで表示します。パスの深い階層も折りたたみ可能なフォルダとして整理されます。

### 🔖 ブックマーク表示
ブックマークもタブと同じツリー形式で閲覧できます。タブ / ブックマークはワンクリックで切り替え可能です。

### 🔍 検索
タブ名・URL・ブックマーク名でリアルタイム検索。ヒット件数がバッジで表示され、該当フォルダが自動展開されます。

### ✕ タブ操作
- **個別閉じ**：タブ横の×ボタンでタブを閉じる
- **一括閉じ**：フォルダ横の×ボタンでグループ内のタブをまとめて閉じる
- **元に戻す**：閉じたタブをUndo機能で復元

### 📌 ステータス表示
ピン留めされたタブは📌、音声再生中のタブは🔊で表示されます。

### 🖱️ 右クリックメニュー
- URLをコピー
- タブのピン留め / 解除
- タブを閉じる
- 新しいタブで開く
- シークレットウィンドウで開く

### ⌨️ キーボード操作
- **↑↓**：項目間の移動
- **←→**：フォルダの開閉
- **Enter**：タブに切り替え / フォルダ開閉
- **Delete / Backspace**：タブを閉じる

### 🌐 多言語対応
英語（デフォルト）と日本語に対応。Chromeの言語設定に自動追従します。

## 使い方

1. 拡張機能アイコンをクリックしてサイドパネルを開く
2. タブ / ブックマークをタブボタンで切り替え
3. フォルダをクリックして展開・折りたたみ
4. アイテムをクリックしてタブに移動 / ブックマークを開く

## インストール

### Chrome Web Store（推奨）
※ 近日公開予定

### 手動インストール（開発者向け）
1. このリポジトリをクローンまたはZIPでダウンロード
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. 展開したフォルダ（`manifest.json`があるフォルダ）を選択

## データについて

すべての処理はブラウザ内で完結します。外部サーバーへの送信は一切行いません。データの保存も行いません。
