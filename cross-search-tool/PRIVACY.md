# Privacy Policy — Cross Search Tool

**Last updated: March 2026**

## Overview

Cross Search Tool is a Chrome extension that generates search links across multiple platforms (Google, Google Maps, Instagram, YouTube, etc.) from a single keyword input, and displays results in a side panel.

## Data Collection

**Cross Search Tool does not collect, transmit, or share any personal data.**

All data created by this extension — including search history, preset configurations, and search results — is stored exclusively in your browser's local storage (`chrome.storage.local` and `chrome.storage.sync`) on your own device.

## What the Extension Accesses

| Access | Purpose |
|---|---|
| Browser local and sync storage | To save and retrieve search history, presets, and current search state |
| New tab creation | To open search result links in new browser tabs |
| Context menu | To enable right-click "Cross Search" on selected text |
| Page content on Google Search, Google Maps, YouTube | To collect search result counts and snippets for the result summary feature (content script) |

## What the Extension Does NOT Do

- Does not send any data to external servers
- Does not track browsing activity
- Does not use analytics or advertising services
- Does not access page content outside of Google Search, Google Maps, and YouTube result pages
- Does not require account creation or login
- Does not collect personally identifiable information

## Data Storage and Deletion

Search history, custom presets, and search state are stored locally in your browser. You can delete search history at any time using the "Clear" button in the side panel.

Uninstalling the extension will remove all associated local data.

## Changes to This Policy

If this policy is updated, the "Last updated" date at the top of this page will be revised. Continued use of the extension after changes constitutes acceptance of the updated policy.
