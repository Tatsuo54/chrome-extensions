# Cross Search Tool

店舗名や製品名をワンクリックで複数プラットフォームに横断検索するChrome拡張機能です。閉業・移転情報のチェックも同時に行えます。

## 主な機能

### 🔍 クロス検索
サイドパネルにキーワードを入力するだけで、Google、Googleマップ、Instagram、X、TikTokなど複数の検索リンクをまとめて生成します。

### ⚠️ 閉業・移転チェック
プリセットに含まれる警告検索（閉業・移転）で、対象の店舗や施設が営業中かどうかを素早く確認できます。

### 📋 プリセット
用途に合わせた検索先セットを切り替えられます。

**ビルトイン（日本語 / English 各5種）：**
- 🔍 一般 / General
- 🍳 料理 / Cooking
- 🍜 飲食 / Restaurants
- 🛒 買物 / Shopping
- ✈️ 旅行 / Travel

**カスタムプリセット：** 自分だけの検索先セットを最大10個作成できます。

### 📖 検索履歴
過去の検索キーワードとプリセットを記録。履歴をクリックするだけで同じ条件で再検索できます。

### 🌐 日英バイリンガル
ブラウザの言語設定に連動してUIが自動で切り替わります。

### 右クリック検索
Webページ上のテキストを選択して右クリック → 「Cross Search」でサイドパネルからクロス検索を開始できます。

## 使い方

1. 拡張機能アイコンをクリックしてサイドパネルを開く
2. プリセットを選択（一般、飲食、料理、買物、旅行など）
3. キーワードを入力して「検索リンクを生成」
4. 個別リンクをクリック、または「すべて新規タブで開く」でまとめて検索

## インストール

### Chrome Web Store（推奨）
※ 近日公開予定

### 手動インストール（開発者向け）
1. このリポジトリをクローンまたはZIPでダウンロード
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択

## データについて

すべてのデータはブラウザのローカルストレージに保存されます。外部サーバーへの送信は一切行いません。詳細は [PRIVACY.md](PRIVACY.md) をご覧ください。
