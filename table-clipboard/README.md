# Table Clipboard

ウェブページ上の表（テーブル）をクリック一つでサイドパネルにストックし、後からCSV、TXT、Markdown形式で一括ダウンロードできるChrome拡張機能です。

## 主な機能

### 🖱️ ワンクリックで表をストック
拡張機能から「現在のページで表を選択」ボタンを押すと、ページ内の表が青枠でハイライトされます。保存したい表をクリックするだけで、サイドパネルにどんどん追加されていきます。
※ページ内に埋め込まれた `iframe` 内の表の取得にも対応しています。

### 💾 複数ページの表をまたいで保持
ストックした表データはブラウザのローカルストレージに保持されるため、別のページに遷移しても消えることはありません。（最大30件まで保存可能）

### 📥 3種類の形式でエクスポート
用途に合わせて以下の形式でデータを出力できます。
- **CSV**（表計算ソフトでの編集に）
- **TXT**（タブ区切りテキスト）
- **MD**（Markdown形式。ドキュメントへの貼り付けに）
個別にダウンロードできるほか、ZIPファイルとしてすべて一括でダウンロードすることも可能です。

### 🌐 日英バイリンガル
ブラウザの言語設定（OSの言語）に連動し、UIが自動で日本語または英語に切り替わります。

## 使い方

1. 拡張機能アイコンをクリックしてサイドパネルを開く
2. 「現在のページで表を選択」ボタンを押す
3. ページ内の青枠で囲まれた表の中から、保存したいものをクリックしてストック（終了する場合は `Esc` キー、またはサイドパネルを閉じる）
4. サイドパネル上でプレビューを確認し、任意の形式でダウンロード

## インストール

### Chrome Web Store（推奨）
※ 近日公開予定

### 手動インストール（開発者向け）
1. このリポジトリをクローンまたはZIPでダウンロード
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択

## データとプライバシーについて

取得した表データはすべてブラウザのローカルストレージに保存されます。外部サーバーへの送信や追跡は一切行いません。安心して機密データの抽出にもご利用いただけます。詳細は [PRIVACY.md](PRIVACY.md) をご覧ください。
