# AI Chat Pin

AIチャットの重要なメッセージをピン留めして、会話を跨いで引き継ぐChrome拡張機能です。

## 対応サービス

- **Claude.ai**
- **ChatGPT**
- **Gemini**

## 主な機能

### 📌 メッセージのピン留め
チャット画面でメッセージにカーソルを合わせると📌ボタンが表示されます。クリックでピン留め・解除ができます。

### 📋 ポップアップで一覧確認
拡張機能アイコンをクリックすると、ピンしたメッセージの一覧が表示されます。

- **コピー**：メッセージ本文をクリップボードにコピー
- **もっと見る / 閉じる**：長いメッセージを展開・折りたたみ
- **メモ追加**：ピンした理由や背景メモを追加・編集
- **削除**：個別のピンを削除

### 🔗 引き継ぎ文の自動生成
ポップアップ下部の「引き継ぎ」セクションで、ピンした内容をもとに次のチャットへの引き継ぎ文を自動生成します。生成された文章は編集でき、自動保存されます。

### 💾 エクスポート
- **TXT形式**：人間が読みやすいテキスト形式
- **CSV形式**：スプレッドシートなどで管理しやすい形式

## 使い方

1. チャット中に重要なメッセージにカーソルを合わせて📌をクリック
2. 必要に応じてポップアップからメモを追加
3. 新しいチャットを始める前にポップアップの「引き継ぎ」→「自動生成」
4. 生成された引き継ぎ文をコピーして次のチャットに貼り付け

## インストール

### Chrome Web Store（推奨）
※ 近日公開予定

### 手動インストール（開発者向け）
1. このリポジトリをクローンまたはZIPでダウンロード
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `pin-extension` フォルダを選択

## データについて

すべてのデータはブラウザのローカルストレージに保存されます。外部サーバーへの送信は一切行いません。
